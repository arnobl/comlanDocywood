### <a name="Defining-a-PropertyAssign"></a>Defining a PropertyAssign

```
thing SendArray {
 	set AnId =
 }

```
Assign a value to a property. `property` must be defined. The expected format is an identifier.
Type `set`. 

Then, give the value, here: `AnId `.


Type `=`.
